# Nova Sonic 管理控制台功能说明

## 概述

Nova Sonic 管理控制台是一个基于React的Web应用程序，用于管理和配置Nova Sonic语音对话系统。该控制台充分利用了python-server的所有后台管理功能，提供了清晰的布局和直观的用户界面。

## 主要功能模块

### 1. 设备管理
**功能描述**：管理和配置所有连接到Nova Sonic系统的设备

**主要特性**：
- 实时显示设备在线状态
- 查看设备基本信息（设备名称、ID、最后连接时间）
- 显示每个设备已启用的工具和服务
- 为每个设备进行独立配置

**设备配置选项**：
- **基本配置**：设备名称、语音模型、系统提示词、模型参数
- **内置工具**：AWS Bedrock知识库、Bedrock Agents、Strands天气代理、传统MCP支持
- **动态MCP服务器**：为设备选择和配置自定义MCP服务器

### 2. 用户管理
**功能描述**：管理系统用户账户，包括管理员和设备用户

**主要特性**：
- 创建新用户（管理员/设备用户）
- 修改用户密码
- 查看用户登录历史
- 用户角色权限管理

**用户角色**：
- **管理员**：可以管理系统和用户，访问所有功能
- **设备用户**：可以连接设备到系统，进行基本操作

### 3. 系统配置
**功能描述**：配置WebSocket连接和全局系统参数

**主要特性**：
- WebSocket服务器连接配置
- 连接状态监控和测试
- 全局默认配置模板
- 事件日志管理

### 4. MCP服务器管理
**功能描述**：管理Model Context Protocol服务器，扩展AI助手能力

**连接类型支持**：
- **STDIO**：本地进程连接，适用于本地工具和脚本
- **SSE**：服务器推送事件，适用于实时数据流
- **HTTP**：流式HTTP连接，适用于Web服务和API

**配置选项**：
- 服务器名称和描述
- 连接参数（命令、URL、参数等）
- 工具名称和功能描述
- 环境变量和HTTP头配置
